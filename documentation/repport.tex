(Q1)
1.
\begin{align*}
    \|Y_l -Y\|_2^2
    &= E[Y_l-Y]^2\\
    &= E[Y_l^2 - 2Y_l Y + Y^2]\\
    &= E[Y_l^2]-2E[Y_l]E[Y]+E[Y^2]\quad\text{ by independence of }Y_l\text{ and }Y\\
    &\to E[Y^2]-2 E[Y]^2 + E[Y^2]\\
    &= 2\underbrace{(E[Y^2]-E[Y]^2)}_{Var[Y]}\\
    &=2 Var[Y] \not\to 0\quad\text{ if }Var[Y]\neq0
\end{align*}

By assumption, $N$ is almost surely finite, i.e. $$P(N=\infty)=0.$$

But $$\{N\ge 1\}\supset \{N\ge 2\}\supset\{N\ge3\}\supset\ldots$$
and  $\cap_{l\ge 1}\{N\ge l\}=\{N=\infty\}.$ so by continuity of probability,
$$0=P(N=\infty)=\lim_{l\to\infty}P(N\ge l), $$ so for $l$ big enough
$$\frac{\|Y_{l-1}-Y\|_2^2}{P(N\ge l)}\ge \frac{Var[Y]}{P(N\ge l)}.$$
But since $P(N\ge l)\to0$ we have $$\frac{Var[Y]}{P(N\ge l)}\to\infty$$ so
$$\sum_{l=1}^{\infty} \frac{\|Y_{l-1}-Y_l\|_2^2}{P(N\ge l)}=\infty$$
2. We want to show

\begin{equation}\label{dummy}
\frac{\|Y_{l-1}-Y\|_2^2}{P(N\ge l)}<\infty.
\end{equation}
By the strong order $p$ assumption and the fact that we have a geometric time step, we have
\begin{align*}
\frac{\|Y_{l-1}-Y\|_2^2}{P(N\ge l)}&\le \frac{\left(C\left(\frac{T}{2^{l-1}}\right)^p\right)^2}{q^l} \\
&= \underbrace{CT^{2p}2^{2p}}_{\tilde{C}}\left(\frac{1}{2^{2p}q}\right)^l.
\end{align*}
In order for this geometric series to converge, we need $\frac{1}{2^{2p}q}<1$, i.e. $q>2^{-2p}.$

We now proceed to show that $\bar{Z}$ has finite variance.
By definition,  $Var[\bar{Z}]=E[\bar{Z^2}]-\left[E\bar{Z}\right]^2.$ But by Jensen's inequality, we have
$$E[\bar{Z}]=E[Y]\le (E[Y^2])^{1/2}<\infty\quad\text{ because }Y\in L^2(\Omega).$$
So we just need to show $E[\bar{Z}^2]<\infty.$
But 
\begin{align*}
    E[\bar{Z}^2]
    &=\sum_{l=0}^{\infty} \frac{\bar{\nu}_l}{P(N\ge l)}\\
    &=\sum_{l=0}^{\infty} \frac{\|Y_{l-1}-Y\|_2^2-\|Y_l-Y\|_2^2}{P(N\ge l)}\\
    &\le\sum_{l=0}^{\infty} \frac{\|Y_{l-1}-Y\|_2^2}{P(N\ge l)}<\infty\quad\text{ by }\ref{dummy}
\end{align*}

1.
\begin{align*}
    E[\bar{\tau}]\\
    &= E\left[\sum_{j=1}^N \bar{t}_j\right]\\
    &= E\left[\sum_{j=1}^{\infty}\bar{t}_j\mathbbm{1}(N\ge j)\right]\\
    &= \sum_{j=1}^{\infty}\bar{t}_jE\left[\mathbbm{1}(N\ge j)\right]\quad\text{ by Tonelli}\\
    &= \sum_{j=1}^{\infty} \bar{t}_jP(N\ge j)
\end{align*}
2. Since $g(\alpha F) = g(F)$,
$$\min_F \underbrace{\left(\sum_{j=0}^{\infty}\frac{\beta_j}{F_j}\right)}_{:=A(F)}\underbrace{\left(\sum_{j=0}^{\infty}t_j F_j\right)}_{:=B(F)}$$
is equivalent to solving
\begin{align*}
    &\min_F &\sum_{j=0}^{\infty}\frac{\beta_j}{F_j}\\
    &\text{subject to } &\sum_{j=0}^{\infty}t_jF_j=C
\end{align*}

indeed if $F^*$ is a minimizer of $A(F)B(F)$, then $F^*$ is a minimizer of
$$\min_F A(F)\text{ s.t. } B(F)=\underbrace{B(F^*)}_{:=C}$$. Indeed, if this is not
the case, there exists $\hat{F}$ such that $A(\hat{F})<A(F^*$) with
$A(\hat{F})=A(F^*)$. But then $A(\hat{F})B(\hat{F})<A(F^*)B(F^*)$ which
contradicts optimality of $F^*$.

This is a constrained optimization problem so we use Lagrange multipliers:
\begin{align*}
\mathcal{L}(F,\lambda)=\sum_{j=0}^{\infty}\frac{\beta_j}{F_j}+\lambda\left(\sum_{j=0}^{\infty}t_j F_j-C\right)\\
\end{align*}

\begin{align*}
\frac{\partial\mathcal{L}}{\partial F_i} &= 0\\
\frac{-\beta_i}{F_i^2}+\lambda t_i&=0\\
F_i &= \frac{1}{\sqrt{\lambda}}\sqrt{\frac{\beta_i}{t_i}}
\end{align*}
In order that $F_0=1$, we need $$\frac{1}{\sqrt{\lambda}}\sqrt{\frac{\beta_0}{t_0}}=1$$
so $\lambda = \beta_0/t_0$, i.e. $$F_i=\frac{\sqrt{\beta_i/t_i}}{\sqrt{\beta_0/t_0}}=:F^*_i.$$
So if $F^*$ is feasible (as stated in the assumption of the question) we have that
$F^*$ satisfies the KKT conditions. Moreover, the problem is strictly convex so $F^*$ is the unique optimal solution.

(Q4)
1. Suppose by contradiction that there is a path with $\tau_{\epsilon}<\infty$. Then by continuity of
$V_t$ there is a time where $V_t$ first crosses $0$. At this point
the process reads $dV_t = \kappa\theta > 0$ which pushes the
process back to a positive value. So the process can never be
negative. But then the original process is also always positive.

2. The idea is to find a Lyapunov function $f(x)$ which explodes
when $x\to 0$, for example $f(x)=x^{-m}$. In order to derive the
dynamics of $f(V_t)$ we use Itô's formula:
$$df(V_t) = f'(V_t)dV_t +\frac12 f''(V_t)\underbrace{d\langle V\rangle_t}_{\sigma^2 V_tdt}$$
with
\begin{align*}
    f'(x) &= - m x^{-m-1}\\
    f''(x) &= m(m+1)x^{-m-2}
\end{align*}

So we have

\begin{align*}
    df(V_t) 
    &= -mV_t^{-m-1}dV_t +\frac12 m(m+1)V_t^{-m-2} \sigma^2V_tdt\\
    &= -mV_t^{-m-1}\kappa (\theta - V_t)dt - mV_t^{-m-1}\sigma \sqrt{V_t}dB_t + \frac12 m(m+1)V_t^{-m-1}\sigma^2dt\\
    &= \left[-m V_t^{-m-1}\kappa(\theta -V_t) + \tfrac12 m(m+1)V_t^{-m-1}\sigma^2\right]dt\\
    &\quad - mV_t^{-m-1}\sigma\sqrt{V_t}\,dB_t\\
    &= mkV_t^{-m}dt -mV_t^{-m-\frac12}\sigma dB_t
\end{align*}

where in the last line we set $m=2\kappa\theta/\sigma^2-1$ in order to simplify
the expression.

Now it is important to note that we cannot apply Itô on all of $[0,t]$ because
$V_t^{-m}$ blows up if $V_t$ hits $0$. But we can write Itô's formula
for the stopped process $$V_{t\wedge\tau_{\epsilon}}\quad\text{where }\tau_{\epsilon}=\inf\{t: V_t\le\epsilon\}.$$
We obtain $$V_{t\wedge\tau_{\epsilon}}^{-m}=V_0^{-m} + \int_0^t m\kappa V_{s\wedge\tau_{\epsilon}}^{-m}ds - \int_0^t m V_{s\wedge\tau_{\epsilon}}^{-m-1/2}\sigma dB_s.$$
Now taking expectation, the term with the Brownian motion disappears because it is a
martingale since $$E[\int_0^{t\wedge\tau_{\epsilon}} V_s^{-2(m+1/2}ds] \le \epsilon^{-2m-1}t<\infty$$
so $$E[V_{s\wedge\tau_{\epsilon}}^{-m}]=V_0^{-m} + mk\int_0^{t}E[V_{s\wedge\tau_{\epsilon}}^{-m}]ds$$ where
we passed the expectation inside the integral using Tonelli's theorem.
By Grönwall's inequality, we then get $$E[V_{s\wedge\tau_{\epsilon}}^{-m}]\le V_0^{-m}e^{mkt}.$$

Now note
\begin{align*}
    P(\tau_{\epsilon}<t) 
    &= P(V_{s\wedge\tau_{\epsilon}}^{-m}\ge\epsilon^{-m})\\
    &le E[V_{s\wedge\tau_{\epsilon}}^{-m}]\quad\text{Markov's inequality}\\
    \le V_0^{-m}e^{m\kappa t}\epsilon^m.
\end{align*}

We now have to show $$P(V_t>0\forall t\in [0,T])=1.$$
This is because
\begin{align*}
    P(\exists t\in[0,T]: V_t\le 0)
    &\le P(\tau_{\epsilon}\le T)\\
    &\le V_0^{-m}e^{m\kappa T}\epsilon^m \to 0\quad\text{when }\epsilon\to0
\end{align*}

So $$P(\exists t\in[0,T: V_t\le0)=0$$ and thus $$P(V_t>0\forall t\in[0,T])=1$$ 

3. Again, we use Itô's formula with
\begin{align*}
g(x) &= x^{1/2}\\
g'(x) &= \frac12 x^{-1/2}\\
g''(x) &= -\frac14 x^{-3/2}
\end{align*}

because $$Z_t = \sqrt{V_t} = g(V_t)$$.
We get
\begin{align*}
    dZ_t
    &= dg(V_t)\\
    &= g'(V_t)dV_t +\frac12 g''(V_t)\underbrace{d\langle V \rangle_t}_{\sigma^2 V_t dt}\\
    &= \frac12 V_t^{-1/2}dV_t -\frac18 V_t^{-3/2} \sigma^2 V_t dt\\
    &=\frac12 V_t^{-1/2}\kappa(\theta-V_t)dt +\frac12 V_t^{-1/2}\sigma V_t^{1/2}dB_t -\frac18 V_t^{-1/2}\sigma^2dt\\
    &= V_t^{-1/2}\left[\frac12 \kappa(\theta-V_t)-\frac18\sigma^2\right]dt + \frac12\sigma dB_t\\
    &= \left[\frac12\kappa(\theta V_t^{-1/2}- V_t^{1/2})-\frac18\sigma^2 V_t^{-1/2}\right]dt +\frac12\sigma dB_t\\
    &= \frac12\kappa \left[\left(\theta-\frac{\sigma^2}{4\kappa}\right)Z_t^{-1}-Z_t\right]dt +\frac12 \sigma dB_t
\end{align*}
